How to Add Portfolio?

To add a portfolio item, navigate to "Add Portfolio" in your user menu, fill in the details, and submit your work to showcase it here.

How to Add Docs?

Use the "Add Docs" option in your user menu to upload documentation files or links relevant to your portfolio pieces.

How to Use Website?

Simply browse the different sections using the navigation bar. Use the search for quick access and explore details with the modals.
